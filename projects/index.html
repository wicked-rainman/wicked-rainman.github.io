Micro dot pHAT for a Raspberry Pi

Recently bought a Micro Dot pHAT to go on a Raspberry Pi. It's driven using I2C and as normal I couldn't find any C code to get the thing going (everyone seems to love python these days!) In the end I grabbed fragments of code I found on the Internet and wrote a set of functions that I think are useful. Help yourself if you were looking for a simple solution.
